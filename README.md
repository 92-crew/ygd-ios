# 여기담 | Your Good Duty

서울시 흡연구역 위치기반 지도 매칭 서비스 애플리케이션

## 프로젝트 개요

---

> ### 여기담(YGD)이란?

- 서울시 흡연구역 및 시설을 나의 위치에서부터 빠르게 찾을 수 있는 편의 서비스

> ### 주요 기능

- Apple Map 을 이용한 지도 서비스
- 서울시 공공데이터 API 를 이용한 지역별 리스팅 및 상세 위치 제공
